# /create-feature-plan

Create a TASK.md file for a new feature using the standard template.

## Steps

1. Read `docs/templates/TASK.md.template`
2. Read the relevant FR from `docs/requirements/functional-requirements.md`
3. Fill in the template with:
   - Feature name and FR reference
   - Acceptance criteria from the FR
   - Technical approach (brief)
   - Files affected
   - Dependencies on other features
4. Save to `docs/tasks/[feature-name].md`
5. Add an entry to `docs/tasks/backlog.md`

Do not start implementation until the task file is reviewed and approved.
